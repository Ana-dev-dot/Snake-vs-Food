## 程序介绍

这是2025年夏季上海交通大学大学面向对象程序设计实践课程大作业

基于SFML图形库的贪吃蛇小游戏

## 编译环境的说明

- Windows 11 专业版
- GCC 8.1.0
- SFML 2.5.1

## 特点

增加了许多新功能

## 操作

点击或长按任意位置，变化蛇的方向

按w/a/s/d或者上/下/左/右键转向

蛇到边界的时候会从对称一边出来

按住空格可以加速
